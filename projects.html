Projects

1. Math Helpline for Newbies

A Java-based desktop application that helps students to learn math in an interactive way. It includes various features such as practice problems, quizzes, and a glossary of mathematical terms.

2. Customer Billing System

A C-based console application that allows businesses to create and manage customer invoices. It includes features such as billing history, customer management, and invoice generation.

3. Hospital Management System

A C-based console application that allows hospitals to manage patient records and appointments. It includes features such as patient registration, appointment booking, and prescription management.

4. Intercity Bus Transport Ally

A C-based console application that provides a real-time schedule of bus arrivals and departures for various intercity bus routes. It includes features such as route planning, ticket booking, and real-time location tracking of buses.

5. Green University Database Management System

A Java-based desktop application that manages the university's academic records, faculty information, and course schedules. It includes features such as student enrollment, faculty management, and course registration.

6. Green University Computer Club Website

A website that showcases the activities and achievements of the Green University Computer Club. It includes information on the club's events, competitions, and workshops, as well as resources for members to learn about programming and technology.
